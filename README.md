# 書目宇宙 GitHub Pages 版

這個資料夾可以直接放到 GitHub repository 的根目錄，然後用 GitHub Pages 發布。

建議 repository 名稱：

```text
Annyeongz
```

發布後網址會像：

```text
https://你的-github-帳號.github.io/Annyeongz/
```

需要的檔案：

- `index.html`
- `data.js`

GitHub Pages 設定：

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`
